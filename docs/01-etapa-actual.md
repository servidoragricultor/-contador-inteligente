# Etapa actual

## Nombre

Etapa 1: ingresos y gastos colaborativos.

## Objetivo

Construir la base operativa para que contador y cliente trabajen sobre la misma empresa.

El cliente captura o importa informacion. El contador revisa, corrige y valida.

## Alcance funcional

- Login.
- Roles simples: contador y cliente.
- Empresas/clientes.
- Invitacion de clientes.
- Registro manual de ingresos.
- Registro manual de gastos.
- Importacion de XML CFDI.
- Deteccion automatica de ingreso o gasto por RFC.
- Deteccion de UUID duplicado.
- Estado financiero basico.
- Estado de revision contable.
- Dashboard simple.
- Exportacion Excel basica.

## Resultado esperado

Un contador debe poder incorporar a un cliente y recibir registros ordenados, con XML procesados automaticamente y con una bandeja clara de informacion pendiente de revisar.

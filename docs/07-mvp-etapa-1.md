# MVP etapa 1

## Funcionalidades incluidas

1. Login.
2. Rol contador.
3. Rol cliente.
4. Empresas/clientes.
5. Invitacion de cliente.
6. Registro manual de ingresos.
7. Registro manual de gastos.
8. Importacion XML CFDI.
9. Deteccion automatica de ingreso o gasto.
10. Deteccion de UUID duplicado.
11. Estados financieros basicos.
12. Revision contable.
13. Dashboard simple por empresa.
14. Exportacion Excel basica.

## Criterio de exito

Un cliente debe poder registrar un ingreso o gasto manual en menos de 10 segundos.

Un cliente debe poder subir XML sin capturar datos fiscales manualmente.

Un contador debe poder identificar rapidamente que registros requieren revision.

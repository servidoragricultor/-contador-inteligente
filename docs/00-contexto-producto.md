# Contexto del producto

Contador Inteligente es una plataforma para contadores, despachos contables y sus clientes.

El objetivo final es convertir documentos fiscales, registros manuales y posteriormente informacion bancaria en datos utiles, revisables y accionables.

El producto no debe comportarse como otro sistema contable tradicional. Debe sentirse como una capa inteligente que reduce captura manual, reduce errores y facilita la comunicacion entre contador y cliente.

## Usuarios principales

- Contador: administra empresas, revisa informacion, corrige registros y exporta datos.
- Cliente: registra ingresos y gastos, sube XML y corrige informacion solicitada por el contador.

## Dolor principal

El contador pierde tiempo solicitando informacion incompleta, ordenando comprobantes, revisando XML, corrigiendo capturas y armando reportes basicos.

El cliente suele registrar tarde, capturar mal o enviar documentos desordenados.

## Propuesta de valor inicial

Permitir que el cliente registre ingresos y gastos de forma muy sencilla, mientras el contador conserva control y revision.

# Instrucciones para agentes IA

## Contexto

Este proyecto es una plataforma para contadores y clientes. La etapa actual es Etapa 1: ingresos y gastos colaborativos.

## Regla principal

No sobreconstruir. La prioridad es que el cliente capture o importe informacion facilmente y que el contador pueda revisar.

## Construir ahora

- Autenticacion.
- Empresas.
- Roles contador y cliente.
- Registros de ingresos.
- Registros de gastos.
- Importacion XML.
- Revision contable.
- Dashboard simple.
- Exportacion Excel.

## No construir todavia

- Conciliacion bancaria.
- Integracion SAT.
- Complementos de pago.
- Polizas contables.
- Reportes avanzados.
- IA avanzada.
- Motor de reglas.

## UX esperada

Lenguaje simple, pocos campos, botones claros y pantallas limpias.

El usuario no debe capturar informacion que ya venga en un XML.

## Criterio de decision

Si una funcionalidad no ayuda directamente a registrar, importar o revisar ingresos y gastos en Etapa 1, debe posponerse.

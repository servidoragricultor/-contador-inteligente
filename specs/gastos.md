# Spec: gastos

## Objetivo

Permitir registrar gastos o compras de forma manual o por XML.

## Registro manual

Campos visibles:

- Monto.
- Fecha.
- Proveedor o descripcion.
- Categoria.
- Estado: pagado o pendiente.
- Nota opcional.

## Registro por XML

Si el RFC receptor coincide con el RFC de la empresa, el XML se registra como gasto.

## Categorias iniciales

- Sin clasificar.
- Renta.
- Servicios.
- Combustible.
- Nomina.
- Honorarios.
- Oficina.
- Transporte.
- Publicidad.
- Comisiones.
- Mantenimiento.
- Inventario o mercancia.
- Impuestos.
- Otro.
